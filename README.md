# HeapSpill

### Live Site
http://HeapSpill.com

## Overview
Inspired by Stack Overflow, HeapSpill seeks to allows users to solve problems encountered while coding. It is built using Ruby on Rails, React on Redux, and Sass.


###Features
* Sign up/in with email
* Ask/Edit questions
* Answer questions
* Vote on answers
* Sort questions
* Fuzzy search
